Splash Screen

The Splash Screen opens by default upon launching Workbench and contains:

á       WB version information

á       links to the HCP website and Twitter feed

á       a shortcut list for opening Specification files (spec files) in the current directory path or those that have been recently opened

To select and open a Spec file listed, click on its title and click the Open button, or double click on its title.

To turn off the default opening of the Splash Screen at startup, turn off this option in Preferences, located in the Workbench Menu (Mac) or File Menu (PC or Linux).
