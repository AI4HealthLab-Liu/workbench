Window Elements Hide/Show Buttons

The Hide/Show buttons for the Workbench Window elements (Information Window, Scenes Window, Toolbar, Overlay Toolbox, and Features Toolbox) are located in the upper right corner of the Workbench Window. These buttons turn the display of these elements on and off. The buttons are shaded gray (as in the images below) to indicate that the element is on, and are unshaded when the element is off.
