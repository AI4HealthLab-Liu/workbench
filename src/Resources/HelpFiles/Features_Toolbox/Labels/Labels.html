Labels

The Labels Tab in the Features ToolBox allows the user to display and view an organized selection list of loaded labels (named colored regions) on brain surfaces.  Therefore, when labels are displayed, regions appear painted on the brain surface. 


Note: Currently, labels are only viewable on surface structures.  Future versions of Workbench will enable this feature in brain volumes as well.

  • Group identifies in which Viewing Tabs the labels will be displayed.  The user can click on each Tab and assign it to a specific Group.  In which case, every Tab assigned to the same Group will display the same labels.
  • Label Attributes allows the user to customize how they want to view the labels.

    • Draw As changes the style of the label components (Filled, Filled and Outline Color, Outline Color, Outline Label Color).
    • Outline Color gives a list of colors to choose from for the label outline.


  • Labels Selection allows the user to select a group of labels or an individual label.
