Foci

The Foci Tab in the Features ToolBox allows the user to display and view an organized selection list of loaded foci on brain surfaces.

Note: Currently, foci are only viewable on surface structures. Future versions of Workbench will enable this feature in brain volumes as well.
  • Group identifies in which Viewing Tabs the foci will be displayed.  The user can click on each Tab and assign it to a specific Group.  In which case, every Tab assigned to the same Group will display the same foci.
  • Display Foci toggles foci on and off
  • Foci Attributes allows the user to customize how they want to view the foci.

    • Contralateral check box toggles foci of both hemispheres onto one hemisphere's surface

    • Paste Onto Surface moves the foci from their coordinate space to directly on the surface displayed

    • Coloring allows the user to select from Name or Class as the color source.

    • Draw As allows the user to change the shape of the foci to Spheres or Squares.

    • Symbol Diameter changes the size of the foci symbols.

  • Foci Selection allows the user to select a group of foci or individual foci.
