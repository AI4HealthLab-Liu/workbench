Borders

 The Borders Tab in the Features ToolBox allows the user to display and view an organized selection list of loaded and newly created borders on brain surfaces.
Note: Currently, borders are only viewable on surface structures. Future versions of Workbench will enable this feature in brain volumes as well.
  • Group: identifies on which Viewing Tabs the borders will be displayed.  The user can click on each Tab and assign it to a specific Group.  In which case every Tab assigned to the same Group will display the same borders
  • Display Borders check box toggles borders on and off
  • Borders Attributes allows the user to customize how they want to view the borders
    • Contralateral check box toggles borders of both hemispheres onto one hemisphere's surface
    • Draw As changes the shape of the border components (Lines, Spheres, Squares, Spheres and Lines)
    • Coloring allows the user to select from Name or Class for the color source
    • Line Diameter changes the border thickness
    • Symbol Diameter changes the size of the sphere or square symbols
    • Unstretched Lines is used when a border from a 3D surface is displayed on a flat surface and the border crosses a surface cut from flattening

  • Borders Selection allows the user to select a group of borders or individual borders
