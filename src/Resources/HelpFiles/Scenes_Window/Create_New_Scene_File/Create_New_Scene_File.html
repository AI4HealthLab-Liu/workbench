Create New Scene File
To create a new Scene File, in a dataset that does not have a current Scene File, first click the Scenes  button in the top right corner of the Workbench window.  The user then needs to set up the Workbench session to display the data in the way they desire to get back to easily.  In the Scenes Window, the user needs to click the New... button.  This is where the Scene File name and location gets set.  Once the name and location is set the Scenes Window will then have only the Add... button activated.
