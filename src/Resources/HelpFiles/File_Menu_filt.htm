File Menu

 

The File Menu appears slightly differently in the Mac version and PC/Linux versions due to Mac-exclusive Workbench Menu. On PC/Linux it contains:

 

·         About Workbench… to view WB version, compiling, and operating system information.

·         Preferences to set user specific Workbench configurations.

·         Mac-specific options available in all applications (Services, Hide Workbench, Hide Others, Show All).

·         Quit Workbench to close the program. 

In the PC and Linux versions, About Workbench…, Preferences, and Exit (to close the program) options appear in the File Menu.
