Select Brainordinate


The Select Brainordinate selection, in the Information Window, is where for any loaded surface you can select a vertex by index number or, if one or more dense connectome CIFTI files are loaded, you can select a File and File Row index to indentify.

  • Surface Vertex

    • Structure is where the user can select on which hemisphere they want to identify a vertex. 

    • Vertex Index allows the user to enter a specific vertex number where they want an ID to appear on the surface.  If the Volume ID button is selected in the Tab section of the Toolbar, the volume will also jump to this specific location.

  • Cifti File Row is only an active option when a DenseConnectome, DenseParcel or Trajectory file is loaded (not all CIFTI files activate this selection)

    • File allows the user to select which file from which to view a particular row.

    • Row Index is the matrix row selection.  This selection will show up as the Map selection in the Overlay ToolBox.
