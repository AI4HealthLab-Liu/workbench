Help Menu

The Help Menu contains information that is meant to assist the user with any questions they have about using Workbench.

  • Search (Mac version only) allows the user to type in a keyword and search for specific information in the wb_view Menus.  For example, type in "tool" and a list of Menu items that include the word tool appear:

 

  • Workbench Help... you're looking at it.  The list on the left are links to general information about Workbench's makeup, such as the Menus. The Find field allows searching for a item within each Help entry.

 

  • Go to HCP Website... will open www.humanconnectome.org in a new browser window.

  • Report a Workbench Bug... is a convenient form the user can fill out and submit if a bug occurs.


  • Submit HCP Software Feature Request... opens http://humanconnectome.org/contact/feature-request.php for an online feature request form.  All suggestions on how to make the HCP Software more robust are welcome.
